## 📖 项目简介

基于 **Vue.js 3** 实现的配置文件格式转换 Web 前端界面，增强了对 **Surge 5+** 的配置格式支持。

> 本项目 fork 自 [CareyWang/sub-web](https://github.com/CareyWang/sub-web)
> 后端核心逻辑源自 [tindy2013/subconverter](https://github.com/tindy2013/subconverter)
>
> 感谢原作者的开源贡献。

## ✨ 特性

- 🎨 基于 Vue 3 + Element Plus 的现代化界面
- ⚡ 采用 Vite 6 构建，极速体验
- 📱 完美响应式设计，支持移动端
- 🔄 实时配置预览与格式转换

## 🔌 后端适配

本项目推荐配合定制后端 **[arcangelw/subconverter-surge](https://github.com/arcangelw/subconverter-surge)** 使用，以解锁以下增强特性：

- **Surge 5+ 深度适配**：支持最新的 Surge macOS/iOS 配置格式 (VIF Mode v3 等)。
- **平台优化**：新增 `surge-macos` 和 `surge-ios` 目标，自动应用最佳实践。
- **增强规则**：内置防污染规则集模板。

> *注：亦可使用标准版 subconverter 后端体验基础功能。*
